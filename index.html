Compliance Event Severity Classification

Applied ML practicum project completed as part of the MS Analytics program at Georgia Tech, using enterprise compliance data retrieved from Snowflake. Built a KNN classifier to predict the severity category of a compliance event at intake, across three possible outcomes, using intake variables including scope, financial impact, and population affected. Applied one-hot encoding for categorical features and limited the training window to three years to account for known criterion shifts over time. Delivered as a working model with a demonstration and findings presentation to a compliance director. Limitations included incomplete data availability at intake and the inherent judgment component in final severity assignment.

Python SQL (Snowflake) KNN Classification Jupyter Georgia Tech Practicum
